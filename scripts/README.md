# 测试数据填充脚本

这个目录包含用于填充测试数据的脚本。

## 活动数据填充脚本

### 功能
为测试旅程自动填充7天的川渝游活动数据，包括：
- 每天安排 3-5 个活动
- 涵盖景点、美食、交通等多种类型
- 行程轻松合理，不会太满
- 景点间距离合理，交通便利
- 从预置POI中挑选的经典景点

### 行程概览
1. **第1天**：成都抵达 + 春熙路 + 火锅
2. **第2天**：大熊猫基地 + 宽窄巷子 + 锦里
3. **第3天**：都江堰 + 青城山一日游
4. **第4天**：成都前往重庆 + 解放碑 + 洪崖洞
5. **第5天**：长江索道 + 磁器口古镇 + 观音桥
6. **第6天**：重庆前往乐山 + 乐山大佛 + 东坡印象水街
7. **第7天**：文殊院 + 天府广场 + 返程

### 使用方法

1. **创建测试旅程**
   - 访问 http://localhost:3000
   - 登录账号
   - 创建一个新旅程（建议设置7天，例如：2025-03-15 至 2025-03-21）

2. **运行填充脚本**
   ```bash
   npm run seed:activities
   ```

3. **查看结果**
   - 脚本会自动找到最新创建的旅程
   - 为每一天填充合理的活动
   - 显示填充进度和结果

### 注意事项
- 脚本会为最新创建的旅程填充数据
- 如果旅程已有活动，脚本会提示并跳过
- 如需重新填充，请先手动删除现有活动
- 确保已正确配置 `.env.local` 文件中的 Supabase 配置

### 活动类型说明
脚本使用的活动类别：
- `attraction` - 景点游玩
- `food` - 美食用餐
- `transport` - 交通出行

### 自定义
如需修改行程安排，编辑 `scripts/seed-activities.js` 文件中的 `activitiesByDay` 数组即可。
